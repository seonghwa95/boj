### 난이도 - 브론즈1

<hr>

### 문제 설명
서로 다른 숫자 n개를 입력받아
오름차순대로 출력하기

### 소스 코드
+ 메모리 : 30840 KB
+ 시간 : 128 ms

```python
n = int(input())
a = []

for i in range(n):
    ali = int(input())
    a.append(ali)

while len(a) != 0:
    print(min(a))
    a.remove(min(a))
```
### 코드설명
+ 숫자 n개를 입력받고 빈 리스트 하나 생성
+ 빈 리스트에 숫자 n개를 전부 추가 해준다.
+ 리스트가 텅 빌때 까지 반복하는데,
+ 그 리스트의 최소값을 출력후에, 최소값을 제거해줌을 반복한다.

### 주요한 점
처음엔 정말 애를 먹었으나 적절한 while문을 사용하니 웬만한 브론즈 2~3 문제 같은 난이도로 느껴졌다. 최소값 출력후 최소값을 지워주는 것이 가장 쉽게 접근한 방법이 아닐까 생각한다.

이렇게 구현을 하면 제한시간을 넘지 않을까? 라고 생각해서 처음엔 다른 방식으로 접근을 했었는데 생각보다 이른 시간에 구현이 되었다.
