### 문제 설명
입력받은 층수의 한층 아래의
1호부터 입력받은 호수 까지의 인원수를 모두 더한 값 출력

### 소스 코드
+ 메모리 : 30840 KB
+ 시간 : 80 ms

```python
def floor(k, n):
    people = []
    zerofl = list(range(1, n+1))

    for m in range(k):
        sum = 0
        firfl = []
        for j in range(n):
            sum += zerofl[j]
            firfl.append(sum)
        zerofl = firfl
        people.append(firfl)
    return people[k-1][n-1]


t = int(input())

for i in range(t):
    k = int(input())
    n = int(input())
    print(floor(k, n))
```
### 코드설명
+ 전체 큰 리스트 안에 리스트를 넣은 2차원으로 구현하였다.
+ 0층의 값들은 미리 설정해둔 후
+ 인원은 누적이기 때문에 반복문 내에서 구한 층의 값을 전층에 다시 할당한다.

### 주요한 점
최초에 재귀를 이용하여 쉽게 구현을 하였으나,
문제에 시간제한이 있었고 시간초과를 하였다.

시간제한에 왠지 Big-O O(n^2) 시간은 시간제한에 걸릴 것 같아서
최대한 O(n) 으로 구현해보려 하였으나,
고민해봐도 떠오르지 않아서 그냥 for문 속에 for문이 들어가도록 구현하였고
그것이 통과되었다.

### 오답코드 (시간초과)

```python
def floor(k, n):
    if k == 0:
        return n
    if n == 1:
        return 1
    return floor(k-1, n) + floor(k, n-1)


t = int(input())
for j in range(t):
    k = int(input())
    n = int(input())
    print(floor(k, n))
```
