### 문제 설명
주어진 막대의 길이에 따라 상자바닥에 온전히 깔리는 길이인지 판단하기

### 아이디어
주어진 막대의 길이가 상자내의 최대길이인 대각선의 길이보다 짧다면
온전히 바닥에 깔릴 수 있다.

### 소스 코드
+ 메모리 : 30840 KB
+ 시간 : 72 ms
```python
n, w, h = map(int, input().split())
dia = w*w + h*h

for i in range(n):
    stick = int(input())
    if dia >= stick*stick:
        print("DA")
    else:
        print("NE")
```
### 코드설명
+ 입력받은 상자의 길이에 대해 대각선**2 의 길이를 구한다.
+ 각 성냥의 길이**2 보다 대각선의 길이가 크거나 같다면 "DA"출력
+ 그렇지 않으면 "NE"출력


### 주요한 점
상자내의 최대길이가 어디인지를 고려하는 것이 가장 주요..
