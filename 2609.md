### 문제 설명
어떤 수 두개를 입력 받아
그 두 수의 최대공약수와 최소 공배수 구하기

### 소스 코드
+ 메모리 : 30840 KB
+ 시간 : 72 ms

```python
a, b = map(int, input().split())

for i in range(max(a, b), 0, -1):
    max_a = a % i
    max_b = b % i
    if max_a == 0 and max_b == 0:
        gr_deno = i
        print(gr_deno)
        break

print(int((a/gr_deno)*(b/gr_deno)*gr_deno))
```
### 사고과정
먼저 최대공약수에 대해 생각해보았다. 공통된 인수를 전부 곱한 것이기 때문에, a와 b 중 큰 수 부터 -1 씩 해 가면서 두 수에 나눴을 때 각각 나누어 떨어지는 가장 큰 수가 최대공약수로 두었다.

최소공배수는 최대공약수x서로소 임을 고려하여

a/최대공약수 X b/최대공약수 X 최대공약수 로 구현하였다.
