### 문제 설명
8진수 n을 입력받아 2진수로 변환하여 출력하기

### 소스 코드
+ 메모리 : 40936 KB
+ 시간 : 96 ms

```python
n = input()
ten = int(n, 8)
bi = list(bin(ten))

bi.pop(0)
bi.pop(0)

print("".join(bi))

```
### 코드설명
+ 숫자 n을 입력받고
+ n(8진수)를 10진수로 변환 /int(n, 8)
+ 10진수를 2진수로 변환하고 리스트로 변경 /bin(10진수)/
+ bin을 사용하면 2진수는 앞에 '0b' 가 붙으므로 pop(0) 두번 실행
+ "".join() 메소드로 리스트를 문자열로 변경후 출력

### 주요한 점
생소한 문법들이 많이 있었다.
엄청 중요하다는 느낌보다는 이런게 있구나 알아두는 정도로 의미를 부여했다.
join 메소드는 기억해두자!! 또한, 메모리가 많이 잡히는 이유에 대해 알아보기
