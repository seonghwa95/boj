### 문제 설명
블랙젝 게임처럼 카드 숫자 3개의 합이 제시한 수와 가장 가깝도록
출력하기!

### 소스 코드
+ 메모리 : 36188 KB
+ 시간 : 108 ms
```python
def black(n, b):
    c = []
    for i in range(n):
        for j in range(i+1, n):
            for k in range(j+1, n):
                num = a[i] + a[j] + a[k]
                if num == b:
                    return num
                elif num < b:
                    c.append(num)
    return(max(c))


n, b = map(int, input().split())
a = list(map(int, input().split()))

print(black(n, b))
```
### 코드설명
+ for 문을 세개를 중첩하여 카드를 순서대로 경우의 수를 만든다.
+ ex) 5장의 카드는 1, 2, 3 ~ 1, 2, 4 ... n-2, n-1, n 카드의 합 전부 구하기
+ 만약 3장의 합이 주어진 수와 같다면 바로 리턴
+ 그렇지 않다면 조건에 따라 주어진 수보다 작은 합만 리스트에 추가
+ 그 리스트의 최댓값을 출력한다.
