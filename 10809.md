### 문제 설명
문자열을 입력받아
각각의 알파벳이 문자열의 인덱스 몇에 일치하는지를
판단하는 문제

### 소스 코드
+ 메모리 : 30864 KB
+ 시간 : 68 ms
```python
inp = list(input())
alpabet = list('abcdefghijklmnopqrstuvwxyz')
for i in alpabet:
    if i in inp:
        print(inp.index(i), end=' ')
    else:
        print(-1, end=' ')
```
### 코드설명
+ 입력받은 문자열과 알파벳을 리스트로 저장
+ 각각의 알파벳이 입력받은 문자열에 속하면 그 첫 인데스를 찾아 출력
+ 문자열에 없는 알파벳은 -1을 출력

### 주요한 점
리스트로 저장하여 리스트.index() 를 통해 첫번째 인덱스를 찾기! 문법
