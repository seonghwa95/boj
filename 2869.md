### 문제 설명
달팽이는 낮에 A만큼 올라가고 밤에 B만큼 떨어진다

### 소스 코드
+ 메모리 : 30864 KB
+ 시간 : 76 ms
```python
A, B, V = map(int, input().split())
x = (V-A)
day = x // (A-B)
if x == 0:
    print(1)
elif day == 0:
    print(day+2)
elif (x % (A-B)) != 0:
    print(day+2)
else:
    print(day+1)
```
