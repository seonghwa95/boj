### 문제 설명
시 분 초 를 입력받고
경과시간(초)를 입력받아서
경과된 후의 시 분 초를 출력하기

### 소스 코드
+ 메모리 : 30864 KB
+ 시간 : 76 ms
```python
a, b, c = map(int, input().split())
d = int(input())

sec = d % 60
mi = (d // 60) % 60
hour = (d // 60) // 60

time_h = a + hour
time_m = b + mi
time_s = c + sec

if time_s >= 60:
    time_m += 1
    time_s -= 60

if time_m >= 60:
    time_h += 1
    time_m -= 60

print((time_h) % 24, time_m, time_s)
```
### 코드설명
+ d(초) 를 시 분 초로 각각 변수를 선언한 후 
+ 초기 시간에 그 시 분 초 를 각각 더한다.
+ 초나 분이 60을 넘기면 분이나 시를 +1씩 해준 후에 초나 분은 -60을 해준다.

### 주요한 점
여러 경우를 판단해서 모든 경우를 다 아우르는 코드를 짜내야 한다.
if 문의 순서또한 주요했다. ex) 아래 if문이 위로 가면 분이 60분이 출력되는 경우가 생긴다.
