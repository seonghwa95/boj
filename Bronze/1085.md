### 코드 설명
+ 직사각형 내 좌표에서 가장 가까운 모서리 좌표로 이동하기
+ 입력 받은 내용에서 리스트로 저장한 후 min() 함수 이용

### 소스 코드
+ 메모리 : 30864 KB
+ 시간 : 76 ms
```python
x, y, w, h = map(int, input().split())
num = [x, y, w-x, h-y]
print(min(num))
```
