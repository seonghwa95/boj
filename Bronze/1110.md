### 코드 설명
+ 일의 자릿 수를 이어붙인 수가
주어진 수가 나오도록 반복
+ while 문을 이용하여 나머지와 몫을 통해 구현

### 소스 코드
+ 메모리 : 30864 KB
+ 시간 : 76 ms
```python
n = int(input())
num = n
count = 0
while True:
    a = num // 10
    b = num % 10
    c = (a+b) % 10
    
    num = 10*b + c
    count += 1
    
    if num == n:
        print(count)
        break
```

### 주요한 점
while문 밖에서 num = n 코드를 이용하여 입력을 반복해서 받지 않도록 한다.

num = 10*b + c 을 이용하여 a와 b 변수에 새로운 num을 할당하여 반복의 결과가 다르게 나오도록 한다.
