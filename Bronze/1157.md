### 문제 설명
대소문 구분없이 문자열을 입력받아 가장 많이 입력받은 알파벳 하나를 대문자로 출력하기!

### 소스 코드
+ 메모리 : 40632 KB
+ 시간 : 448 ms
```python
n = input()
n = list(n.upper())
alpa = list('ABCDEFGHIJKLMNOPQRSTUVWXYZ')
cnt = []
for i in alpa:
    co = n.count(i)
    cnt.append(co)

if cnt.count(max(cnt)) > 1:
    print('?')
else:
    ind_mx = cnt.index(max(cnt))
    print(alpa[ind_mx])
```
### 코드설명
+ 문자열을 입력받아 upper() 로 전부 대문자로 만들고 리스트로 놓기
+ 알파벳 리스트를 만들고 빈 리스트(각 알파벳별 나온 숫자 리스트)를 하나 만든다.
+ for문으로 각 알파벳별 카운트를 빈 리스트에 추가하기
+ 만약 가장 많이 나온 알파벳이 2개 이상이면 ? 출력
+ //cnt.index(max(cnt)) >> cnt 리스트에서 가장 큰값의 인덱스//
