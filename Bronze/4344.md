### 문제 설명
입력받은 횟수를 입력받은 후
학생 수와 각 학생의 점수를 입력받는다.

그리고 각각 평균을 넘은 학생의 비율이 몇인지를 출력한다.

### 소스 코드
+ 메모리 : 30860 KB
+ 시간 : 76 ms
```python
n = int(input())
for i in range(n):
    num_list = list(map(int, input().split()))
    avg = sum(num_list[1:])/num_list[0]
    avg_list = 0
    for j in range(1, len(num_list)):
        if num_list[j] > avg:
            avg_list += 1
    print("{:.3f}%".format(avg_list/num_list[0]*100))
```
### 코드설명
+ i for문에서 리스트로 학생 수와 점수를 입력받은 후
+ 학생 수를 제외한 강의 평균을 1 인덱스 부터 학생수(0인덱스) 로 구한다.
+ j for문에서 조건문을 통해 평균을 넘는 학생 수를 구한다.
+ 문자열 포매팅을 이용하여 %를 함께 출력하는 것과 소수점 3째 짜리까지 출력한다.

### 주요한 점
포매팅을 통해 소수점 출력과 문자열에 %붙여서 출력하기!!
