### 문제 설명
각 문자에 대응되는 숫자를 찾아
문자를 숫자로 바꾸기

### 소스 코드
+ 메모리 : 30864 KB
+ 시간 : 76 ms
```python
inp = list(input())
sec = 0
for i in inp:
    if i in 'ABC':
        sec += 3
    elif i in 'DEF':
        sec += 4
    elif i in 'GHI':
        sec += 5
    elif i in 'JKL':
        sec += 6
    elif i in 'MNO':
        sec += 7
    elif i in 'PQRS':
        sec += 8
    elif i in 'TUV':
        sec += 9
    elif i in 'WXYZ':
        sec += 10
print(sec)
```
### 코드설명
+ 문자를 입력받아서
+ 각 문자별로 숫자를 더해
+ 총합을 출력한다.
