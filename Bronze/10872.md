### 문제 설명
재귀 함수를 이용하여 팩토리얼 구현하기!

### 소스 코드
+ 메모리 : 30864 KB
+ 시간 : 68 ms
```python
def facto_re(n):
    if n <= 1:
        return 1

    return n * facto_re(n-1)


n = int(input())
print(facto_re(n))
```
### 코드설명
+ 함수를 정의 > 0! 과 1! 은 값이 1 이므로 n <= 1 이면 1을 리턴한다.
+ 재귀함수가 진행하다보면 n이 1일 때 return 값이 1 재귀가 종료되고 그 자리가 1로 대체된다.
+ n * 함수(n-1) 재귀로 팩토리얼 구현
