### 문제 설명
직사각형의 세 점을 입력받아서
나머지 한 점 찾기!

### 소스 코드
+ 메모리 : 30860 KB
+ 시간 : 72 ms
```python
a, b = map(int, input().split())
c, d = map(int, input().split())
e, f = map(int, input().split())

t = [a, c, e]
n = [b, d, f]
if t.count(a) == 1:
    g = a
elif t.count(c) == 1:
    g = c
else:
    g = e

if n.count(b) == 1:
    h = b
elif n.count(d) == 1:
    h = d
else:
    h = f

print(g, h)
```
### 코드설명
+ 각 x 좌표를 리스트에 넣어서 리스트중 1개만 있는 값 뽑기
+ 각 y 좌표를 리스트에 넣어서 리스트중 1개만 있는 값 뽑기
+ x, y 좌표 출력!
