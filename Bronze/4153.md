### 문제 설명
세 변의 길이을 입력받아
짖각 삼각형인지 아닌지를 판단한다.

### 소스 코드
+ 메모리 : 30860 KB
+ 시간 : 72 ms
```python
while True:
    a, b, c = map(int, input().split())
    if a + b + c == 0:
        break
    tri = [a, b, c]
    wid = min(tri)
    hyp = max(tri)
    tri.remove(wid)
    tri.remove(hyp)
    for i in tri:
        if i**2 + wid**2 == hyp**2:
            print('right')
        else:
            print('wrong')
```
### 코드설명
+ while 문으로 반복하는데 입력이 0 0 0이면 멈춘다.
+ 입력을 list에 저장하고 가장짧은 변과 빗변을 따로 저장해둔다.
+ 중간 길이의 변과 가장 짧은 변의 제곱의 합이 빗변의 제곱과 같은지 판단한다.

### 주요한 점
나머지 한 변을 어떻게 정수로 연산을 시킬 것인지가 주요했다.
