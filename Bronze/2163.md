### 문제 설명
n, m 칸 짜리 초콜릿을 입력받은 후
1, 1 칸 짜리로 전부 쪼개기 위한
횟 수를 구한다.

### 소스 코드
+ 메모리 : 30864 KB
+ 시간 : 76 ms
```python
n, m = map(int, input().split())
print((n-1)+n*(m-1))
```
### 코드설명
+ n, m 칸을 입력받은 후
+ 수학적 일반항을 구한 후 출력한다.

### 주요한 점
문법이나 구현이 어렵지는 않음!! 수학적 일반항을 구하는게 관건
