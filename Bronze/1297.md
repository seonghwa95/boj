### 문제 설명
대각선의 길이와 높이, 넓이의 비율을 받아
높이, 넓이의 길이를 구하는 문제

### 소스 코드
+ 메모리 : 30864 KB
+ 시간 : 72 ms
```python
D, H, W = map(int, input().split())

print(int(H*(D/((H**2+W**2)**0.5))), int(W*(D/((H**2+W**2)**0.5))))
```
### 코드설명
+ 길이 > 비율*(대각선 길이/루트(높이비율제곱 + 넓이비 제곱))
+ 조건으로 인하여, int로 길이를 정수로 만든다.

### 주요한 점
