### 문제 설명
달팽이는 낮에 A만큼 올라가고 밤에 B만큼 떨어진다

### 소스 코드
+ 메모리 : 30864 KB
+ 시간 : 76 ms
```python
A, B, V = map(int, input().split())
x = (V-A)
day = x / (A-B)
if int(day) != day:
    print(int(day+2))
else:
    print(int(day+1))
```
### 코드설명
+ x > 나무가 다음날이면 정상에 오를 수 있는 높이
+ day > x만큼 올라가는데 걸리는 일 수!
+ int(day) > day의 정수형 ex) int(3.14) > 3
+ 만약 int(day) 와 day 가 다르면 day + 2 (다음날에 정상도착 +1, 소수점도 하루만큼 올라간 것 +1)
+ 그렇지 않으면 day + 1 (다음에날 정상도착 + 1)
