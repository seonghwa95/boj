### 문제 설명
재귀 함수를 피보나치 수 구하기!

### 소스 코드
+ 메모리 : 30864 KB
+ 시간 : 72 ms
```python
def fibo_re(n):
    if n == 1:
        return 1
    elif n == 0:
        return 0

    return fibo_re(n-1) + fibo_re(n-2)


n = int(input())
print(fibo_re(n))
```
### 코드설명
+ 함수를 정의 0번째 피보나치 수 = 0  과 1번째 피보나치 수는 값이 1 이므로 각각의 값을 리턴한다.
+ 재귀함수가 진행하다보면 n이 1일 때 return 값이 1 n이 0일 때 return값이 0 으로 재귀가 종료되고 그 자리가 각 값으로 대체된다.
+ (n-1) + (n-2) 재귀로 피보나치 수 구현
