### 문제 설명
호텔 투숙객 순서대로 방 배치하기! 101호 부터 102~103 ~~~
201 ~ 202 ~~~~ 순서로 배치한다.

### 소스 코드
+ 메모리 : 30840 KB
+ 시간 : 84 ms
```python
t = int(input())
for i in range(t):
    h, w, c = map(int, input().split())
    if c % h == 0:
        print(100*h + c//h)
    else:
        print(100*(c % h) + (c//h+1))
```
### 코드설명
+ 테스트 케이스 수 입력받기
+ 테스트 케이스 수만큼 반복
+ if문 투숙객 총 수를 높이로 나눈 나머지가 0인지 아닌지 구분
+ 0이면 층 수는 꼭대기층, 호수는 총 투숙객 수를 높이로 나눈 몫
+ 아니면 층 수는 총 투숙객 수를 높이로 나눈 나머지, 호수는 총 투숙객 수를 높이로 나눈 몫+1
