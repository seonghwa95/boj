### 코드 설명
+ 퀴즈를 풀어 맞았으면 +1 점 연속해서 맞았다면 +2, +3 ~.. 틀리면 0점
+ 입력 받은 문장을 split('X') 로 나누어 O만 있는 리스트로 나눈다.
+ 리스트의 인덱스별로 range(ren()) 을 통해서 더해준다. 

### 소스 코드
+ 메모리 : 30864 KB
+ 시간 : 84 ms
```python
n = int(input())
for i in range(n):
    ox = input()
    add = 0
    ox_list = ox.split('X')
    for j in range(len(ox_list)):
        if ox_list[j].count('O') == 1:
            add += 1
        elif ox_list[j].count('O') > 1:
            add += sum(range(ox_list[j].count('O')+1))
    print(add)
```

### 주요한 점
X는 0점이니 스플릿으로 O만 있는 list로 나누고

sum.range(count)를 통해 마치 씨그마의 기능을 구현한다.
