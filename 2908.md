### 문제 설명
세자리 숫자 두개를 입력받아 그 각각수의 거꾸로 수중
더 큰 숫자를 출력하기

### 소스 코드
+ 메모리 : 30864 KB
+ 시간 : 68 ms
```python
a, b = input().split()
int_a = ''
int_b = ''
a = list(a)
b = list(b)
a.reverse()
b.reverse()
for i in a:
    int_a += i
for j in b:
    int_b += j
int_a = int(int_a)
int_b = int(int_b)
print(max(int_a, int_b))
```
### 코드설명
+ a, b를 문자열로 입력받음 (숫자지만 문자로 입력받기)
+ 공백 문자열 두개 변수지정
+ a, b의 숫자리스트를 거꾸로
+ for문을 통해 공백 문자열 변수에 a, b 숫자를 연결해놓기
+ 연결시킨 문자열을 정수형으로 변환후 둘 중에 더 큰 값 출력

### 주요한 점
문자열로 지정한 숫자 리스트와 공백 문자열을 할당해놓은 변수에
for문을 통해 연결해서 붙이는 점을 짜내기가 은근 어렵게 느꼈다.
