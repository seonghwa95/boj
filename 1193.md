### 문제 설명
주어진 표와 예시를 토대로
n을 입력받았을 때 표상에서 n번째 분수를 출력하기
ex) n=1, 1/1 | n=2, 1/2 | n=3, 2/1 | n=4, 3/1 ~~...

### 소스 코드
+ 메모리 : 30860 KB
+ 시간 : 72 ms (함수사용)
+ 시간 : 76 ms

```python
def sig(n):
    sum = 0
    total = 0
    while True:
        sum += 1
        total += sum
        if n <= total:
            return sum


def total(n):
    sum = 0
    total = 0
    while True:
        sum += 1
        total += sum
        if n <= total:
            return total


n = int(input())

if sig(n) % 2 == 0:
    print('{}/{}'.format(sig(n)-(total(n)-n), total(n)-n+1))
elif sig(n) % 2 == 1:
    print('{}/{}'.format(total(n)-n+1, sig(n)-(total(n)-n)))
```
### 코드설명
+ sig 함수에는 1+2+3+4 에서 n이 속하는 수를 뽑아냈다. ex)n=10이면 4, n=4이면 3
+ total 함수에는 n이 속하는 부분의 총합을 구했다. ex)n=10이면 10, n=11이면 15
+ n을 입력받은 후 sig가 짝수냐 홀수냐에 따라 분모, 분자가 바뀐 일반항을 만든다.

```python
n = int(input())

i = 0
sum = 0
while sum < n:
    i += 1
    sum += i

if i % 2 == 0:
    denomin = sum - n + 1
    numer = i - (sum-n)
else:
    denomin = i - (sum-n)
    numer = sum - n + 1

print("{}/{}".format(numer, denomin))
```

### 코드설명
+ n을 입력받고 위에 sig와 total 과 같은 개념으로 i 와 sum을 뽑아내어
+ i가 짝수냐 홀수냐에 따라 각각 분모 분자의 일반항을 만든다.
+ 문자열 포매팅을 이용하여 출력한다.

### 주요한 점
