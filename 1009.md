### 문제 설명
테스트 케이스를 입력받아
각 케이스별로 a, b 를 입력받고
a**b 의 일의 자리를 출력한다.
일의 자리가 0이면 10을 출력한다.

### 소스 코드
+ 메모리 : 30840 KB
+ 시간 : 108 ms

```python
t = int(input())

for i in range(t):
    a, b = map(int, input().split())
    aa = a % 10
    bb = b % 4
    b2 = b % 2

    if aa == 0:
        print(10)
    elif aa == 1 or aa == 5 or aa == 6:
        print(aa)
    elif aa == 4 or aa == 9:
        if b2 == 0:
            print(aa**2 % 10)
        else:
            print(aa % 10)
    elif aa == 2 or aa == 3 or aa == 7 or aa == 8:
        if bb == 0:
            print(aa**4 % 10)
        else:
            print(aa**bb % 10)
```
### 코드설명
+ 테스트 케이스를 입력받고 테스트 케이스 만큼 반복
+ a, b 를 입력받고 aa(a의 일의 자리만 보기) 정의
+ bb(aa의 값에 따라 제곱사이클이 4인 경우를 위해), b2(제곱사이클 2인 경우)
+ 각각의 반복문 aa가 0 / aa 제곱이 항상 같은 경우 / aa 의 제곱이 2가지인 경우 / aa의 제곱이 4가지인 경우

### 주요한 점
정말 쉬운 코딩이라고 생각하였으나..
처음에 a**b 로 코딩하였더니 시간초과가 떴다.

후에, 각 경우를 나누어서 숫자가 커지는 것을 막았더니 통과하였다.
제곱연산은 되도록 피하는 것이 주요하다고 느꼈다.
