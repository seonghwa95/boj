### 난이도 - 브론즈1

<hr>

### 문제 설명
문자열을 입력받아 모든 접두사를 정렬한 후에
사전순으로 다시 정렬한다.
사전순으로 정렬한 접두사들이 처음 정렬한 접두사의 몇번째 인덱스 인지 구하기
(인덱스가 끝에서 부터 시작 끝이 0)

### 아이디어
접두사를 차례대로 나열하면
뒤로 갈수록 뒤 알파벳 한개씩만 줄어들기 때문에
사전순서는 그냥 그 나열의 반대가 된다.

### 소스 코드
+ 메모리 : 30864 KB
+ 시간 : 68 ms
```python
s = input()
s_len = len(s)

for i in range(s_len):
    print(i)
```
### 코드설명
+ 반복문 범위를 0 ~ 문자열길이 까지 설정 한후 출력
