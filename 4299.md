### 문제 설명
축구 점수의 합과 차만 알려주었을 때
축구 점수 출력하기

### 소스 코드
+ 메모리 : 30840 KB
+ 시간 : 68 ms

```python
s, m = map(int, input().split())
if s+m < 0 or s-m < 0 or (s+m) % 2 != 0:
    print(-1)
else:
    a = (s + m) // 2
    b = s - a
    print(max(a, b), min(a, b))
```
### 코드설명
+ 합과 차를 입력받은 후
+ 문제의 조건에 맞게 조건문 설정
+ s = a + b   m = a - b 를 이용하여 a, b 구하기

### 주요한 점
이제까지 백준이 입력을 조건에 맞게 알아서 해주는 것 같았는데,
조건을 추가해주니 오답이 아니었다는 것이 놀라웠다..
앞으로 조건을 항상 추가해주는 버릇을 가져야겠다.
