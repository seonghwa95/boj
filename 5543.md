### 문제 설명
햄버거 와 음료 각각 최소값을 구해서

세트가격 합산에 -50 가격 출력

### 소스 코드
+ 메모리 : 30860 KB
+ 시간 : 72 ms
```python
bur1 = int(input())
bur2 = int(input())
bur3 = int(input())
coke = int(input())
spe = int(input())

bur_list = [bur1, bur2, bur3]
dri_list = [coke, spe]

print(min(bur_list)+min(dri_list)-50)
```
### 코드설명
+ 입력 받은 것중 리스트로 만들어서
+ 리스트중 최솟값 두개를 더해 -50

### 주요한 점
간단한 수학 계산 문제였음
