### 문제 설명
테스트 케이스 개수를 입력받은 후
반복할 횟수와 문자열을 입력받는다.
각각의 테스트 케이스에 대한 결과를 출력한다.
ex) 3 ABC -> AAABBBCCC

### 소스 코드
+ 메모리 : 30860 KB
+ 시간 : 72 ms
```python
n = int(input())
for i in range(n):
    a, b = input().split()
    a = int(a)
    b = list(str(b))
    for j in b:
        print(a*j, end='')
    print()
```
### 코드설명
+ 반복할 횟수 a를 int로 입력받은 후
+ 각각의 문자를 출력할 b를 list로 입력받는다.
+ for 문에서 b의 각각의 문자를 a만큼 곱한후 출력한다.

### 주요한 점
print문에서 end=''로 문자열을 이어서 붙여야하며

테스트케이스를 하나 출력했을때 end=''때문에 입력을 줄바꿈없이 받으므로

print()을 마지막에 넣어줌으로서 입 출력의 줄바꿈을 실행했다.
