### 문제 설명
설탕의 무게를 입력받아
3kg과 5kg의 봉투로 담아 배달하기
최소한의 봉투 수를 출력하기
(3kg와 5kg봉투로 다 담을 수 없게 남으면 -1출력)

### 소스 코드
+ 메모리 : 30864 KB
+ 시간 : 72 ms
```python
n = int(input())
if n % 5 == 0:
    print(n//5)
elif (n-3) % 5 == 0:
    print((n-3)//5 + 1)
elif (n-6) % 5 == 0 and (n-6) >= 0:
    print((n-6)//5 + 2)
elif (n-9) % 5 == 0 and (n-9) >= 0:
    print((n-9)//5 + 3)
elif (n-12) % 5 == 0 and (n-12) >= 0:
    print((n-12)//5 + 4)
else:
    print(-1)
```
### 코드설명
+ 무게를 입력받는다.
+ 봉투수를 최소화 하기 위해서 일단 5로 나누어 떨어지는지 확인
+ 3은 총 4번의 사이클을 돌아 5의 배수가 되기때문에 -3, -6, -9, -12를 했을 경우를 생각해서 계산한다.
+ 음수또한 나머지가 양수로 나오기 때문에 각각의 경우에서 음수가 되지 않도록 and 조건을 붙인다. 

### 주요한 점
생각보다 수학적 계산에서 애를 먹었다.

5를 최대한 쓰는 경우를 고려한 것, 3이 최대 4번의 사이클을 통해 5의 배수에 속하게 되는 것,
음수에 나머지를 구하면 양수로 나온다는 것을 고려하여 경우의 수를 처리하는 것이 주요했다.
