### 문제 설명
유클리드 기하학과 택시 기하학에서의 반지름이 주어졌을 때
원의 넓이 구하기!

### 소스 코드
+ 메모리 : 32976 KB
+ 시간 : 68 ms
```python
from cmath import pi


r = int(input())

print('{:.6f}'.format(pi*(r**2)))
print('{:.6f}'.format(2*(r**2)))
```
### 코드설명
+ cmath 로부터 pi 임포트 하기 !! (VSCODE에서 자동으로 해줌..ㄷㄷ)
+ 반지름을 입력받아
+ 포매팅으로 소수점6 자리까지 원의 넓이 각각 출력 !

### 주요한 점
