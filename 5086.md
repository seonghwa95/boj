### 문제 설명
두 수를 입력 받아 첫 번째 수가 두 번째 수의
약수인지 배수인지를 판별하는 문제.

### 소스 코드
+ 메모리 : 30840 KB
+ 시간 : 64 ms

```python
while True:
    a, b = map(int, input().split())

    if a == 0 and b == 0:
        break

    if b % a == 0:
        print('factor')
    elif a % b == 0:
        print('multiple')
    else:
        print('neither')
```
### 코드설명
+ 반복문 세팅 두 수 입력 받기
+ 입력받은 수가 0 0 일 때 까지 반복
+ b 를 a 로 나눈 나머지가 0 이면 약수
+ a 를 b 로 나눈 나머지가 0 이면 배수
+ 둘 다 아니면 neither 출력

### 주요한 점
문제의 조건이 애매해서 틀린 문제였으나
구현은 매우 쉬웠다.
+ 시간 복잡도 $$O(n)$$
