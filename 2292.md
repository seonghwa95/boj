### 난이도 - 브론즈2

<hr>

### 문제 설명
벌집 모양의 숫자집이 있다.
벌집이 한바퀴 둘러쌀 때마다 그에 대응되는 방이 1씩 늘어난다.
가운데는 1 그 주위를 둘러싼 6개는 2 그 주위를 둘러싼 12개는 3 ~~~

### 소스 코드
+ 메모리 : 30864 KB
+ 시간 : 76 ms
```python
n = int(input())
sam = 1
ad = 6
room = 1
while True:
    if sam >= n:
        print(room)
        break
    sam += ad
    ad += 6
    room += 1
```
### 코드설명
+ while반복문으로 sam 에 ad를 더해주고 (+6 +12 +18 ~~~
+ ad 는 계속 6만큼 room은 1만큼 커진다.
+ sam이 n 보다 크거나 같아 질때의 room을 출력한다.

### 주요한 점
간단한 수학 계산 문제였음..
