### 문제 설명
방학기간동안 숙제를 최대한 빨리 하고 놀 수 있는 일 수 구하기

### 소스 코드
+ 메모리 : 30860 KB
+ 시간 : 68 ms
```python
l = int(input())  # 총 일수
a = int(input())  # 국어 페이지
b = int(input())  # 수학 페이지
c = int(input())  # 국어 최대
d = int(input())  # 수학 최대
if a % c == 0:
    korean = a//c
else:
    korean = a//c + 1

if b % d == 0:
    mat = b//d
else:
    mat = b//d + 1
d = max(korean, mat)
print(l-d)
```
### 코드설명
+ 총 일 수, 수학, 국어 숙제분량, 하루에 가능한 국어, 수학 최대 분량을 입력받기
+ 국어 수학의 총 분량을 가능한 최대 분량으로 나눈 나머지가 0인지 아닌지 구하기
+ 0이면 그 몫이 숙제를 하는데 필요한 최소의 일 수이고 0이 아니면 1을 더한 것이 최소 일 수
+ 국어와 수학중 더 오래 걸린 것을 총 일수에서 뺀다.

### 주요한 점
코드적인 문제보단 수학적 구현문제..
