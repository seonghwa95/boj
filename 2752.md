### 문제 설명
서로 다른 숫자 3개를 입력받아
오름차순으로 출력하기

### 소스 코드
+ 메모리 : 30840 KB
+ 시간 : 72 ms

```python
a, b, c = map(int, input().split())
sum = a + b + c
mid = sum - min(a, b, c) - max(a, b, c)
print(min(a, b, c), mid, max(a, b, c))

```
### 코드설명
+ 숫자 3개를 입력받기
+ 최소값은 min 이용
+ 최대값은 max 이용
+ 중간값은 세 숫자의 합 - min - max 로 구현


### 주요한 점
너무나도 쉬운 브론즈 문제
